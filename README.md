# Library Management System

A simple Library Management System built using Python, Object-Oriented Programming (OOP), and JSON file handling.

## Features

* Add Book
* Show All Books
* Search Book by Title
* Delete Book
* Issue Book
* Return Book
* Save Data to JSON File
* Load Data from JSON File

## Technologies Used

* Python
* OOP (Classes and Objects)
* JSON
* Git & GitHub

## Project Structure

```text
library_management_system/
├── book.py
├── library.py
├── main.py
├── books.json
└── README.md
```

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python main.py
```
